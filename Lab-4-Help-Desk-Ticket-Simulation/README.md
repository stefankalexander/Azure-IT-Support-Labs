# Lab 4 — Help Desk Ticket Simulation

## Objective
Simulate common Tier 1 IT support troubleshooting tickets.

## Example Ticket

User reports they cannot access the internet from their workstation.

## Troubleshooting Steps

1. Verify network configuration using ipconfig
2. Test internet connectivity using ping
3. Check DNS resolution using nslookup
4. Trace network route using tracert

## Skills Practiced

- Troubleshooting methodology
- Root cause analysis
- Network diagnostics
- Tier 1 help desk workflow

## Outcome

Simulated a real-world help desk troubleshooting process and resolved connectivity issues using command-line diagnostic tools.
